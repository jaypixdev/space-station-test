OpenGuidebook(Nullable<Dictionary<String, GuideEntry>>, Nullable<List<String>>, Nullable<String>, Boolean, Nullable<String>)

Opens the guidebook.

Declaration
public bool OpenGuidebook(Dictionary<string, GuideEntry>? guides = null, List<string>? rootEntries = null, string? forceRoot = null, bool includeChildren = true, string? selected = null)
Parameters
Type Name Description
System.Nullable<Dictionary<System.String, GuideEntry>> guides

What guides should be shown. If not specified, this will instead raise a GetGuidesEvent and automatically include all guide prototypes.
System.Nullable<List<System.String>> rootEntries

A list of guides that should form the base of the table of contents. If not specified, this will automatically simply be a list of all guides that have no parent.
System.Nullable<System.String> forceRoot

This forces a singular guide to contain all other guides. This guide will contain its own children, in addition to what would normally be the root guides if this were not specified.
System.Boolean includeChildren

Whether or not to automatically include child entries. If false, this will ONLY show the specified entries
System.Nullable<System.String> selected

The guide whose contents should be displayed when the guidebook is opened
